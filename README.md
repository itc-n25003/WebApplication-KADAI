プロジェクト概要
    ブラウザゲーム「艦隊これくしょん」の月間ランキングの現状と過去のボーダーをまとめたサイトです。
機能
    サーバごとにページを分けて、それぞれのデータと外部サイトの該当サーバページを同時に見れるようにしています。
    右側はmicroCMSからデータを取得して表で見れるようにしています
    左側は外部サイトの該当サーバページをiframeで表示しています

工夫した点
    過去のデータが膨大なため、まとめるためにchatGPTを利用したこと
    月ごとにボーダーの傾向があるので月ごとにデータをセレクトできるようにし、初期表示を現在と同じ月に指定したこと
    Iframeの読み込みが他データより時間がかかるため、Iframeのみロード画面を追加したこと
    外部サイトを参照する際のURLもmicroCMSでページごとに変わるようにしたこと
